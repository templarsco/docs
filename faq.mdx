---
title: FAQ
description: Frequently asked questions about the Lightweight API.
---

<AccordionGroup>
  <Accordion title="What models are available?">
    Lightweight provides unified access to 25+ frontier and efficient models from OpenAI, Anthropic, Google, and xAI. See the [Models Catalog](/models) for the complete list, or query `GET /v1/models` for the live catalog.
  </Accordion>
  <Accordion title="Is this OpenAI-compatible?">
    Yes. Lightweight is a drop-in replacement for the OpenAI API. Change your Base URL to `https://api.lightweight.one/v1` and use your Lightweight API key. No code changes needed.
  </Accordion>
  <Accordion title="How do I check my usage?">
    Use the [Dashboard](https://lightweight.one/dashboard) for a visual overview, or query `GET /v1/usage` for programmatic access.
  </Accordion>
  <Accordion title="What happens when I hit my limit?">
    You'll receive a `429 Too Many Requests` HTTP response with a `Retry-After` header indicating how long to wait.
  </Accordion>
  <Accordion title="Can I use streaming?">
    Yes. Add `"stream": true` to your request payload. Responses use Server-Sent Events (SSE) format matching the OpenAI specification.
  </Accordion>
  <Accordion title="How do I get an invite?">
    Lightweight is in invite-only beta. Open a [Discussion](https://github.com/templarsco/lightweight/discussions) in the Access Requests category to request access.
  </Accordion>
</AccordionGroup>
