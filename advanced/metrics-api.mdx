---
title: Metrics API
description: Real-time performance data for all models.
---

Access real-time performance data including latency, throughput, uptime, and token usage. No authentication required.

## Endpoint

```
GET https://api.lightweight.one/api/metrics
```

## Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `period` | string | `24h` | Time range: `1h`, `24h`, `7d`, `30d` |
| `model` | string | — | Filter by model ID |

## Example Requests

```bash
# Default 24h metrics for all models
curl https://api.lightweight.one/api/metrics

# 7-day metrics for a specific model
curl "https://api.lightweight.one/api/metrics?period=7d&model=gpt-4.1"
```

<Note>
  Data refreshes every 30 seconds. Powered by Cloudflare Analytics Engine with 90-day retention. See the live dashboard at [lightweight.one/status](https://lightweight.one/status).
</Note>
