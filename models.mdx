## Model Aliases

For convenience, you can use these short names:

| Alias | Resolves To |
|-------|-------------|
| `claude-opus` | `claude-opus-4.6` |
| `claude-sonnet` | `claude-sonnet-4.6` |
| `claude-fast` | `claude-opus-4.6-fast` |
| `opus-fast` | `claude-opus-4.6-fast` |
| `opus-4.6` | `claude-opus-4.6` |
| `sonnet-4.6` | `claude-sonnet-4.6` |

<Warning>
  **Do not use versioned model names** like `claude-sonnet-4-20250514` or `claude-haiku-4-5-20251001`. Use the short names listed above.
</Warning>

## Live Catalog

Query the API for the always-current list:

```bash
curl https://api.lightweight.one/v1/models \
  -H "Authorization: Bearer lw_sk_your-key-here"
```

See [Models API](/advanced/models-api) for the full response schema.
