## v1.x — Next

### Embeddings API

New `/v1/embeddings` endpoint for generating text embeddings — numerical vectors that capture semantic meaning.

**Use cases:**
- **Semantic search** — Find documents by meaning, not keywords
- **RAG** — Embed docs, retrieve context, feed to LLM
- **Classification** — Group similar content automatically
- **Duplicate detection** — Identify semantically similar items

```bash
# Coming soon
curl https://api.lightweight.one/v1/embeddings \
  -H "Authorization: Bearer lw_sk_your-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-3-large",
    "input": "The quick brown fox jumps over the lazy dog"
  }'
```
