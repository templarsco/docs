## v1.x — Embeddings API

We're adding support for embeddings models through the standard OpenAI-compatible `/v1/embeddings` endpoint. This will let you generate vector embeddings from multiple providers using the same unified API.

**Preview:**

```bash
curl https://api.lightweight.one/v1/embeddings \
  -H "Authorization: Bearer lw_sk_your-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "text-embedding-3-large",
    "input": "The quick brown fox jumps over the lazy dog"
  }'
```

<Note>
  The Embeddings API is currently in development. Join the [community discussions](https://github.com/templarsco/lightweight/discussions) to follow progress and share feedback.
</Note>

## v2 — Agent Skills

Version 2 will introduce **Agent Skills** — a framework for connecting AI models to tools and external services through the Lightweight gateway.

Agent Skills will allow you to:

- Define tool schemas and connect them to any model
- Execute multi-step agent workflows across providers
- Chain models together (e.g., use a fast model for routing and a flagship model for generation)
- Access built-in skills for common tasks like web search, code execution, and file management

More details will be shared as development progresses. Stay tuned on the [Dashboard](https://lightweight.one/dashboard) and [GitHub](https://github.com/templarsco/lightweight/discussions).
